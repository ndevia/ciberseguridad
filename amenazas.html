Amenazas Comunes

El objetivo de un DDoS es colapsar el servidor, el sitio web o la red de destino sobrecargándolo de tráfico.

  • Implantar protocolos de seguridad de red, como cortafuegos y redes privadas virtuales (VPN).
  • Abordar las lagunas o vacíos de su ciberseguridad.

La ingeniería social es el uso de la manipulación y el engaño para obtener acceso no autorizado a sistemas e información. Explota el factor humano de la ciberseguridad.

  • Formar a los empleados para que sean emocionalmente inteligentes y perceptivos.
  • Disponer de procedimientos sólidos de autenticación para el acceso a los sistemas de la empresa.

El tipo más común de ingeniería social, consiste en enviar mensajes falsos a personas de la empresa, con el objetivo que el destinatario divulgue información privada o haga clic en un enlace que permita que el malware acceda a sus cuentas o infecte su sistema informático.

  • Exigir el uso de correos electrónicos corporativos y filtros de spam.
  • Educar a los empleados sobre el aspecto que pueden tener los mensajes falsos de personas clave de la organización.

Los ataques Man-in-the-middle tienen lugar cuando el hacker es capaz de introducirse en un proceso de transmisión de información, como un dispositivo que se conecta a una red o un usuario que se registra en una cuenta.

  • No conectarse a redes Wi-Fi públicas o que no requieran datos de acceso.
  • Visitar sólo sitios web con el protocolo HTTPS.

Un ataque de contraseña es básicamente cualquier intento de obtener la contraseña de alguien sin su consentimiento.

Esto se puede lograr espiando físicamente a alguien mientras introduce su contraseña, adivinar utilizando información disponible públicamente, mediante keyloggers y otros programas espía, entre otros.

  • No utilizar nunca la misma contraseña o variaciones de ésta para varias cuentas.
  • Utilizar contraseñas únicas y fuertes, que no estén relacionadas con datos que se encuentren de manera pública en las redes sociales.

Una inyección de lenguaje de consulta estructurado (SQL) consiste en añadir código malicioso a un servidor utilizando SQL, pudiendo forzar al servidor a liberar información sensible.

  • Resolver las vulnerabilidades SQL con la ayuda de su webmaster o de un especialista en informática.
  • Utilizar la validación de entradas, las consultas parametrizadas o las sentencias preparadas, y los cortafuegos de aplicaciones web.

El malware es un software de ciberataque, diseñado específicamente para ayudar al hacker o grupo de hackers a obtener un acceso no autorizado.

Algunos tipos de malware son: Ransomware, Software de redes de bots, Troyanos, Troyanos de acceso remoto (RATs), Virus y gusanos.

  • Instalar un software antivirus y configurarlo para analizar automáticamente los sistemas.
  • Añadir varias capas de protección para dificultar al máximo el acceso de los piratas informáticos a los activos digitales (cambiar las contraseñas con frecuencia, exigir una autenticación adicional).

El ransomware es un tipo de malware que impide a los propietarios acceder a sus propios sistemas informáticos y archivos. Puede incluir amenazas de borrar datos o hacerlos públicos.

  • Instalar un software antivirus y configurarlo para analizar automáticamente los sistemas.
  • Cambiar las contraseñas con frecuencia.
  • Exigir una autenticación adicional.
